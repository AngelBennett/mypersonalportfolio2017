Using CSS to give the make the pages of my portfolio has been a learning experience of trial and error, like learning html. I'm learning how valuable a tool it is in making visually pleasing web pages. Being able to minimize redundancies by having a central place to refer to a base code that can be used for many pages has been very helpful on this assignment
